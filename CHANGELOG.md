# Changelog

All notable changes to Jumpy Friend.

## [1.2.0] - 2026-02-18

### Fixed
- **High-DPI Display Support** - Render at 2x resolution for crisp graphics on retina displays
- Fixed pixelated graphics on modern displays

---

## [1.1.0] - 2026-02-17

### Added
- **Spin Rotation** - Character spins while jumping for visual flair
- **Air Jump Limit** - Limited to 1 air jump (was unlimited)

### Changed
- **New Character Design** - Green square with black smiley face (removed sprite dependency)
- **Super Powerful Jump** - Velocity range from -600 to -1200 for dramatic jumps

---

## [1.0.0] - 2026-02-16

### Added
- **One-Way Platforms** - Pass through from below, land on top
- Cleaner collision logic

### Fixed
- Jump mechanics (velocity range -450 to -950)
- Removed debug UI

---

## [0.2.0] - 2026-02-15

### Fixed
- Hold-to-charge mechanic reliability
- Charging state detection

---

## [0.1.0] - 2026-02-14

### Added
- **Initial Release**
- Hold-to-jump charging mechanic
- Procedurally generated platforms
- Endless vertical climb gameplay
- Score tracking

---

*A simple hold-to-jump platformer game.*
